# v3.9.1 릴리즈 노트

일자: 2026-02-18

## 주요 변경

- 외부 의존 통합 테스트 게이트를 명시적으로 분리했습니다.
  - `external` 태그 테스트는 `-PincludeExternalIntegration`일 때만 실행됩니다.
- API 통합 테스트의 모듈 경계를 강화했습니다.
  - policy/RAG/output-guard API 통합 테스트와 관련 SQL 스키마를 `arc-web` 소유로 정렬했습니다.
- 유지보수성을 위해 대형 파일 분해를 이어갔습니다.
  - `AgentProperties.kt`에서 정책/기능 속성 클래스를 분리했습니다.
  - `McpManager.kt`에서 `McpToolCallback`을 분리했습니다.
- 포크 초기 설정 가이드를 정리했습니다.
  - provider 키 예시를 Spring AI 환경변수 기준으로 통일했습니다.

## 운영 영향

- 기본 로컬 테스트는 더 빠르고 안정적으로 유지되고, 외부 의존 테스트는 필요할 때만 실행합니다.
- CI에서 웹 API 통합 회귀를 모듈 경계 기준으로 더 명확히 검출합니다.
- 포크 후 확장 시 프로젝트 구조를 이해하고 수정하기가 쉬워졌습니다.

## 업그레이드 메모

- 표준 통합 스위트:
  - `./gradlew :arc-core:test :arc-web:test -PincludeIntegration --tests "com.arc.reactor.integration.*"`
- 외부 의존 통합 스위트(필요 시):
  - `./gradlew test -PincludeIntegration -PincludeExternalIntegration`
- 프로젝트 버전은 `3.9.1`입니다.
