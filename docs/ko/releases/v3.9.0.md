# v3.9.0 릴리즈 노트

일자: 2026-02-17

## 주요 변경

- 모듈 경계에 맞춰 통합 테스트 소유권을 정렬했습니다.
  - policy/RAG/output-guard API 통합 테스트를 `arc-web`로 이동
- CI에 통합 테스트 게이트를 추가했습니다.
  - `:arc-core:test :arc-web:test -PincludeIntegration --tests "com.arc.reactor.integration.*"`
- 로컬 반복 개발 속도 개선을 위해 Gradle 기본값을 추가했습니다.
  - daemon, parallel, build cache, Kotlin incremental
- 외부 의존 통합 테스트 게이트를 분리했습니다.
  - `external` 태그 테스트는 `-PincludeExternalIntegration`일 때만 실행
- 대형 파일 분해를 추가로 진행했습니다.
  - `AgentProperties.kt`에서 정책/기능 설정 클래스를 분리
  - `McpManager.kt`에서 `McpToolCallback` 분리

## 운영 영향

- 이전에 숨겨질 수 있던 API 통합 회귀가 CI에서 즉시 드러납니다.
- 포크 프로젝트에서도 기본 설정만으로 테스트/빌드 루프가 더 빨라집니다.
- 웹 API 통합 테스트가 우연한 클래스패스 상태에 의존하지 않게 됩니다.

## 업그레이드 메모

- 통합 검증 실행:
  - `./gradlew :arc-core:test :arc-web:test -PincludeIntegration --tests "com.arc.reactor.integration.*"`
- 외부 의존 통합 테스트 실행(필요 시):
  - `./gradlew test -PincludeIntegration -PincludeExternalIntegration`
- 릴리즈 인덱스에 `v3.9.0`이 추가되었습니다 (`docs/ko/releases/README.md`).
- 자체 CI를 운영 중이면 동일한 통합 테스트 단계를 반영하세요.
