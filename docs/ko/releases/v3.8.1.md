# v3.8.1 릴리즈 노트

일자: 2026-02-17

## 주요 변경

- MCP 런타임 update/connect 신뢰성 개선
- 기본 로컬 테스트 실행 경로 속도 개선
- 모듈 전반 코루틴 취소 처리 패턴 통일
- 실행 조립 모듈 `:arc-app` 추가
- 문서 구조를 카테고리 기반으로 재편

## 운영 영향

- 잘못된 MCP 설정에서 더 빠르게 실패하여 지연 감소
- 통합 테스트는 `integration` 태그로 opt-in 유지
- MCP 서버 수정 시 runtime manager 상태 동기화 강화

## 업그레이드 메모

- 로컬 실행은 `./gradlew :arc-app:bootRun` 사용
- MCP 원격 전송은 streamable HTTP 대신 SSE 사용 권장
