# v4.5.0

Release date: 2026-02-28

## Highlights

- **Documentation overhaul**: Complete per-module reference documentation covering all 7 modules (arc-core, arc-web, arc-admin, arc-slack, arc-discord, arc-line, arc-error-report) in English and Korean. Each guide covers key components, all configuration properties, extension points, and practical code examples.
- **Admin API reference**: Comprehensive 80+ endpoint reference for the admin control plane — personas, sessions, documents, prompt templates, MCP servers, auth, metrics, tracing, tenants — with full request/response specs and authentication guide (EN+KO).
- **README rewrite**: New README with badges, ASCII architecture diagram, 8-module table, and deployment options (Docker, Helm, local). Added `README.ko.md` Korean translation.
- **CLAUDE.md / AGENTS.md rewrite**: Restructured agent instruction files using best-practice formats. CLAUDE.md trimmed from 270 → 210 lines with no information loss. AGENTS.md reduced to a focused Codex-first format (67 lines) that validates first and defers to CLAUDE.md.
- **Gitleaks config**: Added `.gitleaks.toml` to suppress false positives from curl examples and placeholder credentials in documentation markdown files.

## Compatibility and Migration

- No API changes from `v4.4.0`.
- No configuration changes required.

## Operational Notes

- Documentation is now fully bilingual (EN+KO) under `docs/en/` and `docs/ko/`.
- Admin API reference: `docs/en/admin/api-reference.md`.
