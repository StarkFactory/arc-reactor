# v5.2.0

Release date: 2026-03-16

## Highlights

- **RAG grounding and document management**:
  - RAG documents now propagated as `VerifiedSources` for grounded agent responses, ensuring citations reference actual retrieved content (#323).
  - Document embedding failures return proper error responses instead of silently dropping documents (#321).
  - Content-hash deduplication added to `DocumentController` to prevent duplicate document ingestion (#322).
  - Merge artifact in `DocumentController` resolved from the dedup PR.

- **Post-release stabilization (v5.1.1)**:
  - Merge conflict markers resolved in `SequentialOrchestrator` and `SchedulerController` (#316).
  - Edge case tests added for RAG compressor, tool cache TTL, and citation special characters (#319).
  - Scheduler edge case tests added for timezone templates and tags (#318).
  - Missing config properties added to `application.yml` (#317).
  - Release notes added for v5.0.0 and v5.1.0 (#315).
  - Version badges and CLAUDE.md config reference updated (#314).

## Compatibility and Migration

- **Document deduplication** is enabled by default. Duplicate documents (by content hash) are rejected with a conflict response. Disable with `arc.reactor.rag.dedup-enabled: false` if duplicate ingestion is intentional.
- No breaking API path or configuration key changes from v5.1.0.
