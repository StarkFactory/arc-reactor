# v4.7.6

Release date: 2026-03-05

## Highlights

- **Redis optional runtime hardening**:
  - Redis-backed token revocation now falls back to in-memory when Redis connectivity is unavailable at startup.
  - Redis semantic cache auto-configuration now falls back to Caffeine when Redis is unreachable, avoiding startup failures.
- **Runtime classpath parity fix**:
  - Added Redis starter to `arc-app` runtime so Redis-backed optional features can activate in the default application module.
- **Deployment config parity (Compose/Helm/K8s)**:
  - Added Redis/cache/token-revocation env knobs to Helm chart config surface.
  - Updated K8s baseline config and docs for production-safe token revocation defaults and optional Redis enablement.
