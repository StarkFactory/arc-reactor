# v5.8.1

Release date: 2026-03-16

## Highlights

- **Critical metrics bean fix**:
  - `MicrometerAgentMetrics` could lose to `NoOpAgentMetrics` due to non-deterministic `@ConditionalOnMissingBean` ordering within the same `@Configuration` class. When Spring processed the NoOp bean first, it registered `AgentMetrics` and the Micrometer bean was skipped, resulting in zero agent execution metrics in Prometheus despite `MeterRegistry` being available (#349).
  - `MicrometerAgentMetrics` is now marked `@Primary` and its `@ConditionalOnMissingBean` is removed, ensuring it always wins when `MeterRegistry` is on the classpath (#349).

## Compatibility and Migration

- **Metrics**: Deployments that were silently missing Prometheus agent metrics will now start reporting them. No configuration change needed.
- No breaking API path or configuration key changes from v5.8.0.
