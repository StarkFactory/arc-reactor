# v5.4.0

Release date: 2026-03-16

## Highlights

- **Guard stage ordering fix**:
  - GuardStage `order` values corrected per CLAUDE.md convention (built-in orders 1-5, custom 10+) (#335).

- **Auto-configuration compliance**:
  - `@ConditionalOnMissingBean` added to all 30 auto-configured beans that were missing the annotation, ensuring every bean can be overridden by users (#337).

- **Observability improvements**:
  - Three metric gaps closed in `MicrometerAgentMetrics` for complete coverage of agent lifecycle events (#333).
  - Configurable trace sampling rate added to reduce production tracing overhead (#334).

- **Code quality**:
  - Magic numbers extracted to named constants in `DynamicSchedulerService` for readability (#336).

- **Agent quality**:
  - Agent response quality improved by removing noise from non-workspace conversations (#339).

## Compatibility and Migration

- **Trace sampling**: Sampling is now configurable via `arc.reactor.tracing.sampling-rate` (default `1.0`, full sampling). Reduce for high-traffic production deployments.
- **Bean overrides**: All auto-configured beans now consistently support `@ConditionalOnMissingBean`. Custom beans will take precedence as expected.
- No breaking API path or configuration key changes from v5.3.0.

## Operational Notes

- Version bumped to `5.4.0`.
- Tag `v5.4.0` triggers GitHub Release and Docker publication workflows.
