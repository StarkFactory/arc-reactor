# v3.8.1 Release Notes

Date: 2026-02-17

## Highlights

- Improved MCP runtime update/connect reliability
- Faster test execution path for default local workflows
- Unified coroutine cancellation handling across modules
- New executable assembly module: `:arc-app`
- Docs reorganization into categorized structure

## Operational Impact

- Startup/runtime behavior around invalid MCP configs fails faster.
- Integration tests stay opt-in (`integration` tag pattern).
- Runtime MCP updates now synchronize manager state explicitly.

## Upgrade Notes

- Use `./gradlew :arc-app:bootRun` for local app startup.
- For MCP remote transport, use SSE when streamable HTTP is unavailable.
