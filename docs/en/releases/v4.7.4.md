# v4.7.4

Release date: 2026-03-04

## Highlights

- **Event-loop safety for LLM and tool execution paths**:
  - Blocking operations in the agent execution path are now offloaded from Reactor Netty event-loop threads.
  - This prevents request starvation and timeout cascades under sustained mixed enterprise traffic.
  - Improves tail latency stability and preserves guard-first behavior under load.
- **Tool callback metadata compatibility hardening**:
  - `SpringAiToolCallbackAdapter` now resolves metadata from both modern and legacy method patterns.
  - Supports Spring AI `ToolDefinition.name()/description()/inputSchema()` while keeping fallback compatibility for getter-style implementations.
  - Reduces tool registration ambiguity when callback implementations expose metadata through `ToolDefinition` only.
