# v5.9.0

Release date: 2026-03-16

## Highlights

- **Bug fix: MediaAttachment.equals()**:
  - Operator precedence bug corrected where `&&` bound tighter than `?:`, causing `uri` and `name` fields to only be compared when `data` was null instead of always being compared (#353).

- **Validation fix**:
  - Missing `@Valid` added to `DocumentController.deleteDocuments()` so `@field:NotEmpty` and `@field:Size` constraints on `DeleteDocumentRequest` are actually enforced (#353).

- **Version alignment**:
  - README.md and README.ko.md version badges updated to 5.9.0 (#353).
  - Helm Chart `appVersion` updated from 5.2.0 to 5.9.0 (#353).

## Compatibility and Migration

- **Document deletion validation**: `deleteDocuments()` now enforces `@NotEmpty` and `@Size` constraints. Clients sending empty or oversized deletion requests will receive HTTP 400 instead of silently proceeding.
- No breaking API path or configuration key changes from v5.8.2.

## Operational Notes

- Version bumped to `5.9.0`.
- Tag `v5.9.0` triggers GitHub Release and Docker publication workflows.
