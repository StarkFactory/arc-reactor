# v5.7.0

Release date: 2026-03-16

## Highlights

- **RAG performance optimization**:
  - Adaptive query routing enabled by default, skipping RAG retrieval for simple greetings and non-retrieval queries (#343).
  - Similarity threshold raised from 0.3 to 0.65 to filter irrelevant documents (#343).
  - `topK` reduced from 10 to 5 and retrieval timeout reduced from 5s to 3s for faster responses (#344).
  - Reranking disabled by default (vector similarity sufficient for most queries) (#344).

- **Retry and budget improvements**:
  - Retry initial delay reduced from 1000ms to 200ms for faster error recovery (#344).
  - Per-message structure overhead (20 tokens) added to context trimmer token estimation for more accurate budget accounting (#344).

- **Agent response quality**:
  - Improved RAG citation instructions in system prompt (#343).
  - Default system prompt now distinguishes RAG-grounded answers from general knowledge responses (#343).

## Compatibility and Migration

- **Similarity threshold**: Raised from 0.3 to 0.65. Documents with lower similarity scores are now filtered out. If retrieval recall drops, lower `arc.reactor.rag.similarity-threshold` back toward 0.3.
- **topK**: Reduced from 10 to 5. Set `arc.reactor.rag.top-k: 10` to restore prior behavior.
- **Reranking**: Disabled by default. Re-enable with `arc.reactor.rag.reranking.enabled: true` if retrieval precision is important.
- No breaking API path or configuration key changes from v5.6.0.
