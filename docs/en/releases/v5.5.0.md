# v5.5.0

Release date: 2026-03-16

## Highlights

- **Agent response quality**:
  - Empty "Sources" footer no longer appended to casual greetings and general questions; `VerifiedSourcesResponseFilter` skips the footer when `requiresVerifiedSources` is false (#339).
  - LLM instructed to include a "Sources" section only when the user prompt involves workspace tools (Jira, Confluence, Bitbucket, Swagger, etc.) (#339).
  - Default system prompt improved to discourage fabricated citations and encourage concise, honest responses across `PersonaStore`, `ChatRequestResolutionSupport`, and `DynamicSchedulerService` (#339).

- **Guard stage ordering**:
  - `UnicodeNormalizationStage` order changed from 1 to 0 to resolve conflict with `RateLimitStage` (both were order 1) (#340).

- **Documentation**:
  - Release notes added for v5.2.0, v5.3.0, and v5.4.0 (#340).
  - README version badges updated from 5.1.0 to 5.4.0 (#340).

## Compatibility and Migration

- **UnicodeNormalizationStage order**: Changed from 1 to 0. Custom guard stages at order 0 should be adjusted to avoid conflicts.
- No breaking API path or configuration key changes from v5.4.0.

## Operational Notes

- Version bumped to `5.5.0`.
- Tag `v5.5.0` triggers GitHub Release and Docker publication workflows.
