# v5.6.0

Release date: 2026-03-16

## Highlights

- **MCP registration fix**:
  - SSRF validator in `McpServerController` now respects the `arc.reactor.mcp.allow-private-addresses` configuration. Previously, private IP addresses (localhost, 10.x, 172.x, 192.168.x) were always blocked regardless of configuration, making it impossible to register local MCP servers in development environments.
  - `SsrfUrlValidator.validate()` now accepts an `allowPrivateAddresses` parameter, and `McpServerController` passes the configured value from `AgentProperties`.

## Compatibility and Migration

- **MCP private addresses**: Development environments that previously worked around the SSRF block no longer need workarounds. The `arc.reactor.mcp.allow-private-addresses` flag now functions as documented.
- No breaking API path or configuration key changes from v5.5.0.

- Version bumped to `5.6.0`.
